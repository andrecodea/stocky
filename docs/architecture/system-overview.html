Key Design Decisions

  • Ambient agents (Deep Agents) are event-driven, not user-triggered — Supabase webhooks fire on DB changes, pg_cron schedules daily reports.
  • Chat Q&A uses RAG (not ambient agents) — reactive, requires fast response, pgvector retrieval keeps latency low.
  • Frontend consumes AI results via Supabase Realtime subscriptions — no polling, no agent waiting in HTTP request.
  • FastAPI and AI workers run in Docker on same VPS (Coolify manages containers) — can split into separate services if load requires.
  • RBAC: Operator role = mobile only (inventory ops). Admin role = web dashboard + financials + team management.
