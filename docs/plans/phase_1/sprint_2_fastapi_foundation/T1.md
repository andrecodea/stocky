**Arquivos:**
- Editar: `pyproject.toml` (adicionar dependências)

---

### Conceito: Estrutura modular FastAPI

> Separar routers por domínio (`/auth`, `/produtos`, `/estoque`, `/financeiro`, `/ia`) permite que cada dev trabalhe em módulos sem conflito.  
> Um `app/main.py` central registra os routers.  
> Sem essa separação, todos os endpoints ficam em um arquivo e causam conflitos de merge.

---

- [FastAPI — Bigger Applications](https://fastapi.tiangolo.com/tutorial/bigger-applications/) — routers e estrutura modular
- [`supabase-py`](https://supabase.com/docs/reference/python/introduction) — cliente Python para Supabase

<details>
<summary>💡 Solução — Passo 3</summary>

```python
# app/main.py
from fastapi import FastAPI

app = FastAPI(title="Stocky API", version="0.1.0")

@app.get("/health")
async def health():
    return {"status": "ok"}
```

```bash
uv run fastapi dev app/main.py
# Acesse: http://localhost:8000/health
# Acesse: http://localhost:8000/docs → Swagger UI
```
