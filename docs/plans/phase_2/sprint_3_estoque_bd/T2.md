# Task 2: RLS granular por role

**Estimativa:** ~40 min  
**Dev:** Backend Dev 2  
**O que é e por que existe:**  
A T3 do Sprint 1 criou RLS base. Aqui refinar para que operador possa criar movimentações mas não deletar, e admin tenha controle total.

**Arquivos:**
- Criar: `supabase/migrations/<timestamp>_rls_roles.sql`

---

### O que você precisa fazer

> **Passo 1 — Helper function para verificar role:**

```sql
-- TODO: CREATE OR REPLACE FUNCTION is_admin() RETURNS boolean
--   AS: SELECT EXISTS(SELECT 1 FROM perfis WHERE id = auth.uid() AND role = 'admin')
-- Usar esta função nas políticas para evitar subquery repetida
```

<details>
<summary>💡 Solução — Passo 1</summary>

```sql
CREATE OR REPLACE FUNCTION is_admin()
RETURNS boolean AS $$
  SELECT EXISTS(
    SELECT 1 FROM perfis WHERE id = auth.uid() AND role = 'admin'
  )
$$ LANGUAGE sql SECURITY DEFINER STABLE;
```

</details>

---

> **Passo 2 — Refinar políticas de `produtos`:**

```sql
-- TODO: DROP policy "produtos_admin_write" (criada no Sprint 1) → recriar com is_admin()
-- TODO: separar em políticas específicas: INSERT, UPDATE, DELETE para admin
```

<details>
<summary>💡 Solução — Passo 2</summary>

```sql
DROP POLICY IF EXISTS "produtos_admin_write" ON produtos;

CREATE POLICY "produtos_insert" ON produtos FOR INSERT TO authenticated
  WITH CHECK (is_admin());

CREATE POLICY "produtos_update" ON produtos FOR UPDATE TO authenticated
  USING (is_admin()) WITH CHECK (is_admin());

CREATE POLICY "produtos_delete" ON produtos FOR DELETE TO authenticated
  USING (is_admin());
```

</details>

---

> **Passo 3 — RLS para `lotes`:**

```sql
-- TODO: ENABLE RLS em lotes
-- TODO: SELECT para authenticated, INSERT/UPDATE/DELETE somente is_admin()
```

<details>
<summary>💡 Solução — Passo 3</summary>

```sql
ALTER TABLE lotes ENABLE ROW LEVEL SECURITY;

CREATE POLICY "lotes_select" ON lotes FOR SELECT TO authenticated USING (true);
CREATE POLICY "lotes_admin_write" ON lotes FOR ALL TO authenticated
  USING (is_admin()) WITH CHECK (is_admin());
```

</details>
