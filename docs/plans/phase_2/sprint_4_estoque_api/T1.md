# Task 1: Camada de serviço de estoque ✅

**Status:** Concluído  
**Dev:** Backend Dev 1

## O que foi feito

`backend/services/product_service.py` implementa toda a lógica de estoque como funções Python puras, validadas contra o Supabase:

| Função | Descrição |
|--------|-----------|
| `listar_produtos()` | Lista todos os produtos ordenados por nome |
| `buscar_produto(id)` | Busca produto por UUID |
| `criar_produto(data)` | Insere novo produto |
| `atualizar_produto(id, data)` | Atualiza parcialmente |
| `deletar_produto(id)` | Remove produto |
| `listar_estoque_atual()` | View `estoque_atual` com `abaixo_minimo` calculado |

As queries estão validadas e funcionando — a T2 só precisa wirear estas funções como endpoints FastAPI.
