## Para React Native (pós-POC)

> **Passo 1 — Instalar dependências:**

```bash
npx expo install @supabase/supabase-js @react-native-async-storage/async-storage
npm install axios
```

> **Passo 2 — Configurar cliente Supabase (`mobile/src/lib/supabase.ts`):**

```typescript
import { createClient } from '@supabase/supabase-js';
import AsyncStorage from '@react-native-async-storage/async-storage';

export const supabase = createClient(
  process.env.EXPO_PUBLIC_SUPABASE_URL!,
  process.env.EXPO_PUBLIC_SUPABASE_ANON_KEY!,
  { auth: { storage: AsyncStorage, autoRefreshToken: true, persistSession: true, detectSessionInUrl: false } }
);
```

> **Passo 3 — Criar cliente axios com interceptor de JWT (`mobile/src/lib/api.ts`):**

```typescript
import axios from 'axios';
import { supabase } from './supabase';

export const api = axios.create({
  baseURL: process.env.EXPO_PUBLIC_API_URL, // http://10.0.2.2:8500 (emulador Android)
});

api.interceptors.request.use(async (config) => {
  const { data } = await supabase.auth.getSession();
  if (data.session?.access_token) {
    config.headers.Authorization = `Bearer ${data.session.access_token}`;
  }
  return config;
});
```

```env
EXPO_PUBLIC_SUPABASE_ANON_KEY=<anon_key>

```
