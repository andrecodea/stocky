## O que foi feito

Navegação multi-página via `st.navigation` + `st.Page` com estado compartilhado via `session_state`:

**`frontend/app/app.py`:**
```python
pages = {
    "": [
        st.Page("pages/1_dashboard.py", title="Dashboard", icon=":material/dashboard:"),
        st.Page("pages/2_estoque.py",   title="Estoque",   icon=":material/inventory_2:"),
        st.Page("pages/3_chat.py",      title="Chat IA",   icon=":material/smart_toy:"),
    ]
}
pg = st.navigation(pages)

# Estado global inicializado antes de pg.run() — persiste entre páginas
if "chat_history" not in st.session_state:
    st.session_state.chat_history = []
if "recomendacoes" not in st.session_state:
    st.session_state.recomendacoes = None

pg.run()
```

**Componentes equivalentes entregues:**
- `st.metric` → KPI cards com delta colorido
- `st.dataframe` + `.style.apply` → tabela de estoque com highlight de alertas
- `st.chat_message` → interface de chat com histórico
- `st.warning` / `st.info` → badges de alerta
