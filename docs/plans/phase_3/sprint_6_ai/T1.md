### Conceito: OpenRouter como proxy LLM

> OpenRouter expõe uma API compatível com OpenAI, mas roteia para modelos de diferentes provedores.  
> LangChain usa `ChatOpenAI` com `base_url="https://openrouter.ai/api/v1"` para conectar.  
> Sem isso, seria necessário criar um cliente diferente para cada provedor de modelo.

- [`langchain-openai` ChatOpenAI](https://python.langchain.com/docs/integrations/chat/openai/) — parâmetro `base_url`

```python
# scratch/ai_proto.py
from langchain_openai import ChatOpenAI
from langchain_core.messages import HumanMessage
from app.config import settings

llm = ChatOpenAI(
    base_url="https://openrouter.ai/api/v1",
    api_key=settings.openrouter_api_key,
    model=settings.openrouter_model_chat,
)

response = llm.invoke([HumanMessage(content="Olá, responda com 'conectado'")])
print(response.content)
```
